CONTROL DE ACCESO TIPO COMERCIAL E INDUSTRIAL

Los sistemas de control de acceso deben cumplir con el objetivo de mantener el control del ingreso y/o salida de personas a las instalaciones de las empresas como también restringir el acceso a las áreas sensibles dentro de ella. Además, los sistemas de control de acceso permiten registrar accesos a las distintas áreas de la empresa, permitiendo siempre mantener un registro de la ubicación del personal en caso de emergencias.

  • Controladoras para sistemas de control de acceso.
  • Lectoras de huella dactilar, identificación de personal rápida con solo un dedo.
  • Lectores de proximidad para sistemas de identificación y control de acceso para sus trabajadores con tarjeta de proximidad.
  • Teclado para control de acceso restringido en áreas mediante teclado con códigos de acceso.
  • Detectores de objetos metálicos.
  • Brazos hidráulicos.
